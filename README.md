## [PRESENTATION EN FRANCAIS]
### 1. <ins>Description Générale</ins>
Project par D-TheProgrammer

Ce projet est un jeu de plateforme en 2D réalisé avec Godot. Vous incarnez un personnage que j'ai créé nommé _"Mister M"_. Mister M est un employé vivant dans le Sous-Monde, un endroit peuplé de monstres maudits appelés _"Y-mens"_ et contrôlés par des contrats de feu. Après avoir décidé de ne plus faire partie de ce système depuis qu'un nouveau patron a pris le contrôle des lieux, Mister M a décidé de se rebeller. Depuis, il est pourchassé par ces Y-mens, des monstres qu'il peut éliminer en leur sautant sur la tête, les libérant ainsi de leurs contrats qui prendront feu et se détruiront.

### 2. <ins>Utilisation</ins>
Dans ce jeu, l'utilisateur sera accueilli par un menu principal qui lui permettra d'accéder aux crédits, aux commandes, de quitter ou de jouer au jeu.   
Lorsque la partie démarre, vous pouvez vous déplacer dans l'environnement et ramasser des pièces. Pour éliminer les ennemis, qui tuent le personnage en un seul coup, vous devez leur sauter sur la tête. Si le personnage meurt à cause des ennemis ou en tombant dans le vide, vous pouvez choisir de recommencer le niveau ou de quitter la partie. Pendant la partie, vous pouvez également mettre le jeu en pause. Dans ce menu de pause, vous avez la possibilité de recommencer le niveau, de retourner au menu principal ou de quitter le jeu.

Pour jouer les commandes sont les suivantes :

- **Z / Haut / ESPACE** : Sauter
- **Q / GAUCHE** : Se déplacer à gauche
- **D / DROITE** : Se déplacer à droite
- **P** : Pour faire pause  

> [!TIP]
> Pour utiliser l'exécutable sur Linux/Ubuntu, téléchargez le dossier "linux-ubuntu_executable" et décompressez-le. Ensuite, lancez-le avec cette commande :
> ```bash
> ./Mister_M.x86_64
> ````
> Pour utiliser l'exécutable sur Windows, téléchargez le dossier "windows_executable" et décompressez-le. Ensuite, lancez-le avec cette commande :
> ```bash
> ./Mister_M.x86_64.exe
> ````

## [ENGLISH PRESENTATION]
### 1. <ins>General Description</ins>
Project made by D-TheProgrammer

This project is a 2D platform game made with Godot. You play as a character I created named _"Mister M"_. Mister M is an employee living in the Underworld, a place populated by cursed monsters called _"Y-mens"_ and controlled by fire contracts. After deciding not to be part of this system anymore since a new boss took control of the place, Mister M decided to rebel. Since then, he has been pursued by these Y-mens, monsters he can eliminate by jumping on their heads, thus releasing them from their contracts which will catch fire and destroy them.

### 2. <ins>Usage</ins>
In this game, the user will be greeted by a main menu that will allow them to access the credits, controls, quit, or play the game.
When the game starts, you can move around the environment and collect coins. To eliminate enemies, which kill the character in one hit, you need to jump on their heads. If the character dies because of enemies or falls into the void, you can choose to restart the level or quit the game. During the game, you can also pause. In this pause menu, you have the option to restart the level, go back to the main menu, or quit the game.

To play the game the controls are as follows:

- **Z / Up / SPACE**: Jump
- **Q / LEFT**: Move left
- **D / RIGHT**: Move right
- **P** : Pause the game

> [!TIP]
> To use the executable on Linux/Ubuntu, download the "linux-ubuntu_executable" folder and unzip it. Then, launch it with this command:
> ```bash
> ./Mister_M.x86_64
> ````
> To use the executable on Windows, download the "windows_executable" folder and unzip it. Then, launch it with this command:
> ```bash
> ./Mister_M.x86_64.exe
> ````
